# Политика в отношении обработки персональных данных

**Дата последнего обновления: 7 июня 2026 г.**

## 1. Общие положения

1.1. Настоящая Политика в отношении обработки персональных данных (далее — Политика) разработана в соответствии с Федеральным законом от 27.07.2006 № 152-ФЗ «О персональных данных» (далее — 152-ФЗ) и определяет порядок обработки персональных данных пользователей мобильного приложения «Wake on LAN» (далее — Приложение).

1.2. Оператором персональных данных является разработчик Приложения (далее — Оператор). Настоящая Политика применяется ко всем процессам обработки персональных данных, осуществляемым Оператором с использованием Приложения.

1.3. Скачивая, устанавливая и используя Приложение, пользователь выражает безоговорочное согласие с настоящей Политикой и предоставляет согласие на обработку своих персональных данных на основании п. 1 ч. 1 ст. 6 152-ФЗ (согласие субъекта персональных данных на обработку его персональных данных). Согласие считается предоставленным с момента начала использования Приложения. В случае несогласия с условиями Политики пользователь обязан прекратить использование Приложения.

1.4. Оператор обеспечивает конфиденциальность персональных данных и реализует требования 152-ФЗ, в том числе в части:
- определения состава обрабатываемых персональных данных;
- принятия правовых, организационных и технических мер по обеспечению безопасности персональных данных;
- обеспечения прав субъектов персональных данных.

## 2. Правовые основания обработки персональных данных

2.1. Правовыми основаниями обработки персональных данных являются:
- Конституция Российской Федерации;
- Федеральный закон от 27.07.2006 № 152-ФЗ «О персональных данных»;
- Федеральный закон от 07.07.2003 № 126-ФЗ «О связи»;
- локальные нормативные акты Оператора в области обработки и защиты персональных данных;
- согласие пользователя на обработку персональных данных (п. 1 ч. 1 ст. 6 152-ФЗ).

2.2. Обработка персональных данных осуществляется исключительно в целях обеспечения функционирования Приложения и исполнения его базового функционала — отправки Wake-on-LAN (WOL) magic-пакетов в локальной сети.

## 3. Состав обрабатываемых персональных данных

3.1. Приложение осуществляет обработку следующих данных:
- MAC-адреса устройств, добавленных пользователем в список сохранённых устройств;
- наименования устройств, присвоенные пользователем соответствующим MAC-адресам;
- IP-адрес и порт целевого устройства, указанные пользователем.

3.2. Приложение НЕ обрабатывает:
- фамилию, имя, отчество;
- адрес электронной почты;
- номер телефона;
- паспортные данные;
- данные банковских карт;
- биометрические персональные данные;
- сведения о состоянии здоровья;
- иную информацию, позволяющую прямо или косвенно идентифицировать личность пользователя (ст. 3 152-ФЗ).

3.3. Приложение не требует регистрации, создания учётной записи или авторизации.

## 4. Цели обработки персональных данных

4.1. Обработка персональных данных осуществляется в следующих целях:
- обеспечение базового функционала Приложения — отправка WOL magic-пакетов в локальной сети;
- сохранение списка устройств пользователя в локальном хранилище устройства;
- проверка состояния сетевого подключения.

4.2. Приложение не осуществляет:
- передачу персональных данных третьим лицам;
- трансграничную передачу персональных данных;
- сбор данных с использованием файлов cookie или аналогичных технологий;
- использование систем аналитики или отслеживания.

## 5. Порядок и условия обработки персональных данных

5.1. Обработка персональных данных осуществляется с использованием средств автоматизации. Все данные хранятся исключительно на устройстве пользователя (на стороне Приложения).

5.2. Хранение данных осуществляется с применением шифрования AES-256 (встроенный механизм `flet.security.encrypt`).

5.3. Обработка персональных данных осуществляется:
- с согласия субъекта персональных данных (п. 1 ч. 1 ст. 6 152-ФЗ);
- без передачи данных оператору или третьим лицам.

5.4. Приложение не осуществляет сбор персональных данных в информационных системах Оператора. Все обрабатываемые данные остаются на устройстве пользователя.

5.5. Оператор не осуществляет автоматизированную обработку персональных данных с целью принятия решений на основании исключительно автоматизированной обработки (профилирования).

## 6. Сроки обработки и хранения персональных данных

6.1. Персональные данные хранятся на устройстве пользователя до момента удаления пользователем соответствующей записи через интерфейс Приложения (смахивание или нажатие кнопки удаления).

6.2. При удалении Приложения все обрабатываемые данные безвозвратно удаляются с устройства пользователя.

6.3. Оператор не устанавливает дополнительных сроков хранения персональных данных, так как данные не передаются на серверы Оператора.

## 7. Права субъекта персональных данных

7.1. Пользователь имеет право:
- на получение информации, касающейся обработки его персональных данных (ст. 14 152-ФЗ);
- на уточнение (обновление, изменение) персональных данных путём редактирования информации в Приложении;
- на уничтожение персональных данных путём удаления отдельных записей или удаления Приложения (ст. 21 152-ФЗ);
- на отзыв согласия на обработку персональных данных в любой момент путём удаления Приложения (ст. 9 152-ФЗ);
- на обжалование действий или бездействия Оператора в уполномоченный орган по защите прав субъектов персональных данных (Роскомнадзор).

7.2. При удалении Приложения обработка персональных данных прекращается, а данные уничтожаются.

## 8. Меры по обеспечению безопасности персональных данных

8.1. Оператор принимает следующие меры по обеспечению безопасности персональных данных:
- применение шифрования AES-256 для хранения данных на устройстве;
- недопущение передачи данных по незащищённым каналам связи;
- минимизация состава обрабатываемых данных (обрабатываются только данные, необходимые для функционирования Приложения).

8.2. Передача данных при отправке WOL-пакета осуществляется исключительно в локальной сети пользователя и не направляется на внешние серверы.

## 9. Используемые разрешения

9.1. Для функционирования Приложения запрашиваются следующие разрешения Android:
- **INTERNET**: необходимо для отправки UDP широковещательных пакетов в локальной сети;
- **ACCESS_NETWORK_STATE**: необходимо для проверки состояния сетевого подключения.

9.2. Указанные разрешения используются исключительно для реализации базового функционала Приложения.

## 10. Трансграничная передача персональных данных

10.1. Приложение не осуществляет трансграничную передачу персональных данных.

## 11. Обработка персональных данных несовершеннолетних

11.1. Приложение не предназначено для лиц младше 14 лет. Оператор не осуществляет целенаправленный сбор персональных данных несовершеннолетних.

## 12. Изменение Политики

12.1. Оператор оставляет за собой право вносить изменения в настоящую Политику. Новая редакция Политики вступает в силу с момента её размещения в Приложении, если иное не предусмотрено новой редакцией Политики.

12.2. Актуальная версия Политики всегда доступна по адресу: https://github.com/yourusername/wol_apk

## 13. Обратная связь

13.1. Все вопросы, предложения и запросы, связанные с настоящей Политикой, могут быть направлены через Google Play Store.

13.2. Оператор рассматривает обращения субъектов персональных данных в срок, не превышающий 30 (тридцати) календарных дней с момента поступления обращения, в соответствии с требованиями ст. 20 152-ФЗ.
